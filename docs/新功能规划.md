# 桌面客户端做不到的新功能

## 基础设施

### AI API 接入配置

**难度**: 简单  
**优先级**: 高（后续 AI 功能的前置依赖）

在设置页面增加 AI 配置项：
- API 类型选择（OpenAI 兼容 / 自定义）
- API Base URL
- API Key
- 模型名称
- 开关：启用/禁用 AI 功能

配置存储在 `.env` 中：`AI_API_URL`、`AI_API_KEY`、`AI_MODEL`、`AI_ENABLED`

启用后，解锁以下 AI 驱动功能。

---

## AI 驱动功能（依赖 AI API 配置）

### 对话摘要

**难度**: 中等  
**依赖**: AI API

对长对话生成摘要：
- 会话顶部显示"AI 摘要"按钮
- 生成最近 N 条消息的要点总结
- 支持选择时间范围生成摘要

### 智能搜索

**难度**: 中等  
**依赖**: AI API

- 自然语言搜索："上周和张某聊了什么关于项目的事"
- AI 理解意图后转化为关键词搜索 + 时间范围过滤

### 消息情感分析

**难度**: 中等  
**依赖**: AI API

- 对话情绪趋势图（按天/周）
- 标记情绪转折点
- 单条消息情感标注

## 数据分析与可视化

### 年度/月度聊天报告

**难度**: 简单  
**优先级**: 高

类似网易云年度报告风格：
- 消息总量趋势图（按月）
- 最活跃时段热力图（24h × 7天）
- 高频词 Top 20 词云
- 最亲密的人 Top 5（按消息量、互动频率）
- Emoji 使用统计 Top 10
- 最长连续聊天记录
- 凌晨聊天次数
- "年度关键词"——出现频率增长最快的词

### 多会话时间线

**难度**: 简单

选择某一天或某个时间段，按时间线展示所有会话的消息流：
- 纵向时间轴，左右分栏显示不同会话
- 可还原"那天发生了什么"
- 支持筛选特定会话参与时间线

### 关键词趋势图

**难度**: 简单

- 输入一个关键词，展示它在不同时间段出现的频率变化
- 折线图 + 时间轴
- 支持多个关键词对比

### 聊天关系图

**难度**: 中等

群聊互动分析：
- 谁回复谁最多（回复关系网络）
- 群内活跃度排行变化
- 派系分析：谁和谁互动更密切

## 实用工具

### 消息书签

**难度**: 简单

给重要消息加星标/标签：
- 消息右键菜单增加"收藏"选项
- 单独的收藏页面，按标签分类浏览
- 支持添加文字备注
- 数据持久化到 `data/bookmarks.json`

### 文件管理器

**难度**: 中等

聚合所有聊天中的文件：
- 按类型分类（图片/视频/文档/其他）
- 按时间排序，按发送者过滤
- 缩略图预览
- 批量下载

### 全局搜索增强

**难度**: 中等

扩展现有搜索功能：
- 正则表达式搜索
- 按发送者过滤
- 按消息类型过滤（仅图片/仅文件/仅链接）
- 搜索结果按会话聚合分组
- 搜索结果导出
